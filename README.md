이 저장소(repository)는 **『금융경제학: 파이썬을 활용한 금융시장과 통화정책의 이해』**  교재에 수록된 그림과 실증 분석을 **독자가 직접 재현(replicate)하고 확장**할 수 있도록 만든 파이썬 코드 모음입니다. 

이 저장소의 목적은 단순히 책 속의 결과를 다시 그려보는 데 있지 않습니다. 독자들이 **데이터의 표본기간을 바꾸고**, **변수를 추가·제거하고**, **분석 가정을 수정**하면서 금융경제학적 직관을 스스로 형성하는 데 도움을 주는 것을 목표로 합니다.

이 저장소는 다음과 같은 용도로 활용될 수 있습니다.

- 학부 화폐금융론 수업
- 대학원 자산가격결정이론 및 매크로-파이낸스 수업
- 학생 개인 프로젝트
- 실증 분석 연습 및 연구 아이디어 탐색

교수자와 학생 모두  자유롭게 수정·확장하여 사용하기를 권장합니다.

경제학적 직관은  **이론만으로도, 데이터만으로도 완성되지 않습니다.** 이 저장소가 독자 여러분이 직접 데이터를 다루고, 관련 읽을거리를 통해 현실과 연결하며, 금융경제학을 “이해하는 눈”을 기르는 데 작은 도움이라도 되기를 바랍니다.

책이나 코드에 있는 오류, 개선사항, 의견 등은 financialeconomicspython@gmail.com으로 보내 주세요. 고맙습니다.

## 저장소 구조

```
FinancialEconomicsPython/
├── README.md              ← 저장소 메인 설명 (교재 안내, 사용법)
│
├── chapters/
│   └── 각 장(chapter)에서 사용된 그림과 분석을 재현할 수 있는 Jupyter Notebook (.ipynb)
│
├── utils/
│   ├── nber_utils.py
│   ├── plot_utils.py
│   └── preamble_core.py
│
└── readings/
    └── README.md: 각 장의 논의와 관련된 뉴스 기사, 블로그 글, 보고서, 추가 읽을거리
```

- **chapters/**  
  각 폴더는 교재의 한 챕터에 대응하며, 해당 장에 등장하는 주요 그림과 분석을 재현하는 `.ipynb` 파일을 포함하고 있습니다.

- **utils/**  
  데이터 불러오기, 그래프 그리기, 날짜 처리 등 여러 장에서 반복적으로 사용되는 코드를 모아 둔 폴더입니다.  
  독자는 이 폴더의 코드를 모두 이해하지 않아도 각 장의 노트북을 실행할 수 있습니다.

- **readings/**  
  각 장에서 다루는 주제와 관련된 **신문 기사, 정책 보고서, 블로그 글, 참고할 만한 추가 읽을거리**를 정리해 둔 폴더입니다.  
  교재의 내용을 현실의 금융시장, 정책 논의, 최근 이슈와 연결해 이해하는 데 목적이 있습니다.  
  **관심 있는 독자가 맥락을 확장해 읽어볼 수 있는 보조 자료**입니다

## 이 저장소를 사용하는 방법 (중요)
파이썬을 처음 접하는 독자도 쉽게 사용할 수 있도록 **Google Colab 사용을 권장**합니다.

대부분의 노트북은 다음과 같은 순서로 구성되어 있습니다.
1. 필요한 패키지 설치
2. 데이터 불러오기
3. 그림 및 실증 분석 재현


### 방법 1: GitHub의 ipynb 파일을 Google Colab에서 실행하기 (권장)

1단계: 실행하고 싶은 노트북 선택
1. 이 GitHub 저장소에서 `chapters/` 폴더로 이동합니다.
2. 실행하고 싶은 `.ipynb` 파일을 클릭합니다.

2단계: Google Colab에서 열기
1. Google Colab에 접속합니다.  
   https://colab.research.google.com
2. 상단 메뉴에서  
   **파일 → 노트북 열기 → GitHub** 를 선택합니다.
3. 저장소 이름(`FinancialEconomicsPython`) 또는  
   실행하고 싶은 노트북의 GitHub URL을 입력합니다.
4. 목록에서 원하는 `.ipynb` 파일을 선택하여 엽니다.

3단계: 노트북 실행하기
노트북의 셀(cell)을 **위에서부터 차례대로 실행**합니다.
- 단축키: `Shift + Enter`

### 방법 2: 파일을 다운로드하여 직접 실행하기 (고급 사용자)

로컬 환경에서 파이썬을 사용하는 독자는 다음과 같이 사용할 수 있습니다.

1. 이 저장소를 `git clone` 하거나 ZIP 파일로 다운로드합니다.
2. Jupyter Notebook 또는 JupyterLab에서 `.ipynb` 파일을 실행합니다.
3. `utils/` 폴더가 같은 디렉토리에 있어야 합니다.

이 방법은 파이썬 환경 설정에 익숙한 독자에게만 권장됩니다.
